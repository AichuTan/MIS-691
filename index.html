Horizon Hotels DSS (Decision Support System)

Course Context

This project was developed as the final assignment for MIS 691: Decision Support Systems. The goal was to design a complete Decision Support System (DSS) to support a real-world decision-making scenario for a specific organization. I selected Horizon Hotels and focused on using AI to deliver personalized guest experiences at scale.

System Goals

  • Increase guest satisfaction and repeat bookings
  • Empower staff with real-time, data-driven service recommendations
  • Unify guest data from multiple platforms through integrated analytics

Key Features

  • Guest scoring model based on preferences, loyalty tier, and sentiment
  • Recommendation engine using collaborative and content-based filtering
  • Role-specific dashboards for frontline staff, marketing, and executives
  • Sentiment analysis using NLP on guest reviews and chat transcripts

Technologies Used

Microsoft SQL Server, Python (scikit-learn, Pandas), Weka, Power BI, RESTful APIs
